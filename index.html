Road Condition Classification
This is the project developed for my final year project. The condition of the road is analyzed using Accelerometer and gyroscope that is installed on the bike. The data is sent to the raspberry pi module which sends the collected data to the cloud which classification ML algorithms particularly SVM will be applied to get the road condition and determine if it is good or bad on a scale of 1 to 5. The image shows the variation of the 3 axis with respect to time . This data is furthur analysed to classify the road and also capture the co-ordinates on the map as well!

Predict number of COVID-19 patients on a given day


In this the number of patients that India will record could be found out. As we know the growth of the number of Covid-19 is exponential so the model that I created was using Polynomial regression with a degree of 5. The model predicts the number very close with an error rate of just 50 to 100 patients on any given day. The image is the actual prediction of the model given.
